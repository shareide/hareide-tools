Klar til opplasting

Claude har generert forbedret SEO-tekst for alle valgte produkter.
Last ned CSV-filen og importer den direkte til Shopify.

```
To filer lastes ned:
  1. hareide-shopify-NO-[dato].csv — Last denne opp i Shopify. Inneholder norsk SEO-tekst på alle produkter.
  2. hareide-shopify-EN-referanse-[dato].csv — Oppbevar denne. Inneholder engelsk tekst klar til bruk når du aktiverer Shopify Markets.

Slik importerer du NO-filen til Shopify:
  1. Gå til Shopify Admin → Products
  2. Klikk Import øverst til høyre
  3. Last opp NO-CSV-filen
  4. Velg "Overwrite existing products"
  5. Klikk Import — Shopify oppdaterer alle produkter automatisk

Kun produkter som ble vellykket behandlet av Claude er inkludert i nedlastingen.

```
